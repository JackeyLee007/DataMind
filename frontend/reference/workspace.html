# 按产品线汇总 Q4 销售数据 import pandas as pd df = pd.read_csv('sales_2024_q4.csv') summary = df.groupby('product_line').agg( revenue=('amount', 'sum'), orders=('order_id', 'nunique'), customers=('customer_id', 'nunique'), profit_margin=('profit', 'mean') ).round(2) # 计算环比和同比 summary['qoq'] = summary['revenue'].pct_change() summary['yoy'] = summary['revenue'].pct_change(4) summary.sort_values('revenue', ascending=False)
🎯 优化建议
1. 咨询服务线（紧急）：建议将该产品线与云服务打包，作为增值服务而非独立产品。当前亏损主因是交付成本过高（人均交付 23 天），可通过标准化交付流程降低 40% 成本。

2. 数据分析线（优化）：利润率偏低是因为免费试用用户占比 67%。建议将免费试用从 14 天缩短为 7 天，并增加使用引导，预计可将转化率从 12% 提升至 18%。

3. 企业方案线（关注）：虽然环比微降 2.1%，但大客户续约率高达 92%。建议加大 upsell 力度，向现有客户推广高级功能模块。
